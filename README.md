# wnmp_batch
nginx(openresty)+mysql+redis+php(php-fpm) On windows

---

### 项目说明

&nbsp;&nbsp;作为一名后台狗，生产环境是nginx([openresty][]) + php + mysql + [memcached][]/redis
平时在Windows下开发，很难随身带一个Lnmp系统，又想方便的调试，故有此项目。
